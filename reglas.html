RN-001 Disponibilidad de alojamientos Un usuario solo puede reservar un alojamiento si hay disponibilidad en el sistema. Reservas de alojamiento El sistema debe verificar la disponibilidad antes de confirmar la reserva. Si no hay disponibilidad, se ofrecerán fechas alternativas. No aplica si el alojamiento permite sobreventa. Políticas de reserva de hoteles Alta Creación: 01/03/2024 - Última actualización: 05/03/2024
RN-002 Límite de reservas por usuario Un usuario no puede tener más de 5 reservas activas al mismo tiempo. Gestión de reservas Se verifica la cantidad de reservas activas del usuario. Si se alcanza el límite, se bloquean nuevas reservas hasta completar las anteriores. No aplica para usuarios con membresía premium. Política de gestión de usuarios Media Creación: 02/03/2024 - Última actualización: 06/03/2024
RN-003 Control de pagos pendientes No se permite hacer nuevas reservas si el usuario tiene pagos pendientes. Gestión de pagos El sistema verifica el estado de pagos antes de permitir nuevas reservas. Si hay pagos pendientes, se bloquean nuevas reservas y se notifica al usuario. No aplica si el usuario tiene un plan de financiamiento aprobado. Política financiera de la plataforma Alta Creación: 01/03/2024 - Última actualización: 07/03/2024
RN-004 Cancelaciones fuera de plazo No se permite cancelar una reserva menos de 24 horas antes del check-in. Reservas Se verifica la fecha actual con la fecha del check-in. Si está fuera del plazo, no se permitirá la cancelación. No aplica si la política del alojamiento lo permite. Términos y condiciones del servicio Media Creación: 03/03/2024 - Última actualización: 08/03/2024
RN-005 Modificación de itinerarios Un usuario puede modificar su itinerario hasta 48 horas antes del inicio del viaje. Planificación de viajes Se verifica la fecha actual con la fecha del viaje. Si está dentro del tiempo permitido, se permite la modificación. No aplica si la reserva incluye boletos de avión no reembolsables. Política de modificaciones de ViajaPlus Alta Creación: 04/03/2024 - Última actualización: 08/03/2024
RN-006 Descuentos por anticipación Los usuarios que reserven con al menos 60 días de anticipación recibirán un 10% de descuento. Reservas y promociones Se valida la fecha de reserva en comparación con la fecha del viaje. Si cumple el criterio, se aplica el descuento automáticamente. No aplica en temporadas altas. Política de descuentos de la empresa Media Creación: 02/03/2024 - Última actualización: 09/03/2024
RN-007 Máximo de viajeros por reserva Una reserva solo puede incluir un máximo de 10 personas. Gestión de reservas Se verifica la cantidad de personas antes de confirmar la reserva. Si se excede el límite, se debe dividir en múltiples reservas. No aplica si el usuario tiene una cuenta corporativa. Reglamento de reservas grupales Alta Creación: 05/03/2024 - Última actualización: 09/03/2024
RN-008 Restricción de edad para ciertos destinos Algunos destinos solo pueden ser reservados por usuarios mayores de 18 años. Selección de destinos Se verifica la edad del usuario antes de confirmar la reserva. Si el usuario es menor de edad, se bloquea la reserva. No aplica si el usuario viaja con un adulto responsable. Normativa de seguridad de ViajaPlus Alta Creación: 06/03/2024 - Última actualización: 10/03/2024
RN-009 Tiempo límite para completar una reserva Una reserva se mantiene activa en el carrito por máximo 30 minutos sin pago. Procesos de reserva Se mide el tiempo transcurrido desde que el usuario inicia la reserva. Si excede el tiempo, se libera la disponibilidad. No aplica para reservas con pago inmediato aprobado. Política de reservas de ViajaPlus Media Creación: 07/03/2024 - Última actualización: 10/03/2024
RN-010 Uso de códigos promocionales Un código de descuento solo puede usarse una vez por usuario. Promociones y descuentos Se valida si el usuario ya utilizó el código previamente. Si ya lo usó, el sistema bloquea la aplicación del descuento. No aplica si el código es parte de una campaña de uso ilimitado. Política de promociones y marketing Media Creación: 08/03/2024 - Última actualización: 11/03/2024
